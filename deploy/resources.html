Inspiration

GitHub uses jquery-pjax and the History API to traverse a code repository tree. Clicking on folders or files transitions content from left to right when drilling down, and right to left when surfacing back up the project tree. This elegant addition reinforces the concept of heiarchy and organization, and provides visual queues that direct our attention to priorty content.
